## Section 2

- Tables

### Code Blocks

To create a code block, simply indent your code with four spaces or one tab:

    def hello_world():
        print("Hello, world!")

### Tables

To create a table, use the following syntax:

| Name | Age | Gender |
| ---- | --- | ------ |
| John | 25  | Male   |
| Jane | 30  | Female |

### Task Lists

To create a task list, use the following syntax:

- [x] Learn Markdown
- [ ] Build a website
- [ ] Profit!
